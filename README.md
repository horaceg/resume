# Horace Guy's resume

See it in action on mobile or desktop at [cv.horaceg.xyz](https://cv.horaceg.xyz).

Largely inspired by https://github.com/chmduquesne/resume.chmd.fr .

## Setup

Install [pixi](https://pixi.prefix.dev/latest/). 

Tasks are defined in [pixi.toml](pixi.toml)

Run tasks with `pixi run <TASK>`, e.g. `pixi run build`.
